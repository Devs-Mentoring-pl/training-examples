# Algorytmika - Przykłady

Implementacje algorytmów ze wszystkich 7 modułów szkoleniowych z algorytmiki.

## Pliki

| Plik | Moduł | Opis |
|------|-------|------|
| `01_binary_search.py` | Szk. 1 | Wyszukiwanie binarne O(log n) |
| `02_binary_search_game.py` | Szk. 1 | Gra w zgadywanie z binary search |
| `03_recursion.py` | Szk. 2 | Potęgowanie, silnia, Fibonacci z memoizacją |
| `04_euclidean_gcd.py` | Szk. 2 | Algorytm Euklidesa (NWD/GCD) |
| `05_primes.py` | Szk. 3 | Pierwszość O(sqrt(n)) + Sito Eratostenesa |
| `06_sorting.py` | Szk. 4 | Bubble, Insertion, Merge, Quick Sort |
| `07_linked_list.py` | Szk. 5 | Lista wiązana jednokierunkowa |
| `08_stack_queue.py` | Szk. 5 | Stos (LIFO) i kolejka (FIFO) |
| `09_graph_adjacency_list.py` | Szk. 6 | Graf - lista sąsiedztwa (klasy) |
| `10_graph_adjacency_matrix.py` | Szk. 6 | Graf - macierz sąsiedztwa |
| `11_dfs_bfs.py` | Szk. 7 | DFS, BFS, najkrótsza ścieżka, cykle, sortowanie topologiczne |

## Uruchomienie

```bash
python 01_binary_search.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
