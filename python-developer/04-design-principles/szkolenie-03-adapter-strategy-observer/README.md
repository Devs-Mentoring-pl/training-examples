# Szkolenie 3: Wzorce projektowe - Adapter, Strategy, Observer

Przykłady kodu z materiałów szkoleniowych dotyczących trzech wzorców projektowych.

## Pliki

- `01_adapter.py` - Adapter (strukturalny): konwersja napięcia - gniazdko europejskie 220V -> urządzenie amerykańskie 110V
- `02_strategy.py` - Strategy (behawioralny): wymienne algorytmy sortowania (BubbleSort / QuickSort)
- `03_observer.py` - Observer (behawioralny): system zdarzeń z subskrypcjami (Email, Log, Slack)

## Uruchomienie

```bash
python 01_adapter.py
python 02_strategy.py
python 03_observer.py
```

## Wymagania

Tylko biblioteka standardowa Pythona (stdlib).
