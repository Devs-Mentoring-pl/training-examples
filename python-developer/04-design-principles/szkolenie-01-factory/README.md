# Szkolenie 1: Wzorce projektowe - Factory Method i Abstract Factory

Przykłady kodu z materiałów szkoleniowych dotyczących wzorców kreacyjnych.

## Pliki

- `01_factory_method_messages.py` - Factory Method: system komunikatów o błędach (HardwareMessage / SoftwareMessage)
- `02_factory_method_bank_accounts.py` - Factory Method: konta bankowe (osobiste, firmowe, oszczędnościowe)
- `03_abstract_factory_gui.py` - Abstract Factory: komponenty GUI dla Unix / Windows

## Uruchomienie

```bash
python 01_factory_method_messages.py
python 02_factory_method_bank_accounts.py
python 03_abstract_factory_gui.py
```

## Wymagania

Tylko biblioteka standardowa Pythona (stdlib).
