# Szkolenie 2: FastAPI - Pydantic i walidacja danych

Standalone snippety demonstracyjne do szkolenia o Pydantic v2 i walidacji danych w FastAPI.

## Uruchomienie

```bash
pip install -r requirements.txt
```

Przyklady 01-04 to czyste skrypty Pydantic - uruchom przez `python 01_basic_models.py`.
Przyklad 05 to aplikacja FastAPI - uruchom przez `uvicorn 05_response_model:app --reload`.
