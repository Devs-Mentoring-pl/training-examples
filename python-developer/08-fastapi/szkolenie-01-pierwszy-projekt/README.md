# Szkolenie 1: FastAPI - Wstep i pierwszy projekt

Projekt demonstracyjny do szkolenia FastAPI - struktura projektu z APIRouter, schematy Pydantic, endpointy GET z fake database.

## Uruchomienie

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Dokumentacja API: http://localhost:8000/docs
