# Szkolenie 2 - Dekoratory

Przykłady kodu z materiału szkoleniowego dotyczącego dekoratorów w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_classmethod_factory.py` | `@classmethod` jako metoda fabrykująca |
| `02_first_decorator.py` | Tworzenie pierwszego dekoratora z `@` |
| `03_decorator_with_args.py` | Dekorator z argumentami - walidacja |
| `04_universal_decorator.py` | Uniwersalny dekorator z `*args, **kwargs` |
| `05_menu_stars_decorator.py` | Dekorator rysujący ramkę z gwiazdek |
| `06_wraps_decorator.py` | `@wraps` - zachowanie metadanych funkcji |

## Uruchomienie

```bash
python 01_classmethod_factory.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
