# Szkolenie 9 - Dataclasses i Enum

Przykłady kodu z materiału szkoleniowego dotyczącego dataclasses i Enum w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_dataclass_basics.py` | `@dataclass` vs zwykła klasa - eliminacja boilerplate |
| `02_field_and_defaults.py` | `field()`, `default_factory`, mutowalne wartości domyślne |
| `03_frozen_and_post_init.py` | `frozen=True`, `__post_init__`, `InitVar`, `order=True` |
| `04_enum_basics.py` | `Enum`, `IntEnum`, `auto()`, iteracja, porównywanie |
| `05_enum_flag_and_practical.py` | `Flag`, maszyna stanów, Enum+Dataclass, `HttpStatus` |

## Uruchomienie

```bash
python 01_dataclass_basics.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej (Python 3.7+).
