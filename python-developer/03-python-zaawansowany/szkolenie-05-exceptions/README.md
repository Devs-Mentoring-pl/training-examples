# Szkolenie 5 - Exceptions

Przykłady kodu z materiału szkoleniowego dotyczącego obsługi wyjątków w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_try_except_basics.py` | Blok `try/except` - podstawowa obsługa wyjątków |
| `02_raise_exceptions.py` | Ręczne rzucanie wyjątków z `raise` |
| `03_try_except_else_finally.py` | Pełny blok `try/except/else/finally` |
| `04_custom_exceptions.py` | Tworzenie własnych wyjątków + kalkulator |

## Uruchomienie

```bash
python 01_try_except_basics.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
