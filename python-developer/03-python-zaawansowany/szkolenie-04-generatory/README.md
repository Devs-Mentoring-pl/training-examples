# Szkolenie 4 - Listy składane i Generatory

Przykłady kodu z materiału szkoleniowego dotyczącego list składanych i generatorów w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_list_comprehensions.py` | Listy składane - kwadraty, filtrowanie, spłaszczanie macierzy |
| `02_generator_basics.py` | Generatory z `yield`, `next()`, `StopIteration` |
| `03_generator_expressions.py` | Wyrażenia generatorów vs listy - porównanie pamięci |
| `04_generator_send_throw_close.py` | Zaawansowane metody: `send()`, `throw()`, `close()` |
| `05_generator_pipeline.py` | Łączenie generatorów w pipeline - Fibonacci |

## Uruchomienie

```bash
python 01_list_comprehensions.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
