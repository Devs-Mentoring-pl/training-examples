# Szkolenie 8 - Type Hints

Przykłady kodu z materiału szkoleniowego dotyczącego typowania w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_basic_type_hints.py` | Podstawowa składnia - zmienne, argumenty, wartości zwracane |
| `02_complex_types.py` | Typy złożone: `list`, `dict`, `tuple`, `set`, zagnieżdżenia |
| `03_optional_union_callable.py` | `Optional`, `Union`, `Callable`, `TypeVar` |
| `04_classes_type_hints.py` | Type hints w klasach, `ShoppingCart`, `@dataclass` |
| `05_advanced_types.py` | `TypedDict`, `Protocol`, `Literal`, `Final` |

## Uruchomienie

```bash
python 01_basic_type_hints.py
```

Wymaga Python 3.10+ (syntax `str | int`). Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
