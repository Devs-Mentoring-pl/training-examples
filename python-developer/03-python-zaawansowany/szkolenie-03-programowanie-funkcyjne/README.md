# Szkolenie 3 - Programowanie funkcyjne

Przykłady kodu z materiału szkoleniowego dotyczącego programowania funkcyjnego w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_lambda_basics.py` | Lambdy - funkcje anonimowe (1-arg, bezarg, wieloarg) |
| `02_lambda_higher_order.py` | Lambda jako argument funkcji higher order |
| `03_filter.py` | `filter()` - filtrowanie elementów wg warunku |
| `04_map.py` | `map()` - transformacja elementów, wiele iterables |
| `05_reduce.py` | `reduce()` - redukcja sekwencji do jednej wartości |

## Uruchomienie

```bash
python 01_lambda_basics.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
