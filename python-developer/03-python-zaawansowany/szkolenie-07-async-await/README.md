# Szkolenie 7 - Async/Await

Przykłady kodu z materiału szkoleniowego dotyczącego programowania asynchronicznego w Pythonie.

## Pliki

| Plik | Opis |
|------|------|
| `01_sync_vs_async.py` | Porównanie synchronicznego vs asynchronicznego podejścia |
| `02_asyncio_gather.py` | `asyncio.gather()` - równoczesne pobieranie danych |
| `03_create_task.py` | `asyncio.create_task()` - zadania w tle |
| `04_event_loop.py` | Event loop w akcji - kolejność zakończenia zadań |

## Uruchomienie

```bash
python 01_sync_vs_async.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
