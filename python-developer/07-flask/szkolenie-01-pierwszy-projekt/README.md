# Flask – Pierwszy projekt (Szkolenie 1)

Dwa podejścia do tworzenia aplikacji Flask: prosty plik i Application Factory z Blueprintami.

## Uruchomienie

### Sposób I – prosty plik
```bash
pip install flask
python app.py
```

### Sposób II – Blueprinty (zalecany)
```bash
pip install flask
export FLASK_APP=flask_project
flask --debug run
```

Aplikacja: http://localhost:5000
