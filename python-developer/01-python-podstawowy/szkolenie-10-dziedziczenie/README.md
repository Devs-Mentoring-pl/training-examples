# Szkolenie 10 - Dziedziczenie

Przykłady kodu z materiału szkoleniowego dotyczącego dziedziczenia i przeładowania operatorów.

## Pliki

| Plik | Opis |
|------|------|
| `01_polygon_inheritance.py` | Dziedziczenie - Polygon i Triangle z wzorem Herona |
| `02_rpg_game.py` | Gra RPG - dziedziczenie z rozszerzeniem metod |
| `03_multiple_inheritance_mro.py` | Dziedziczenie wielokrotne i MRO |
| `04_operator_overloading.py` | Przeładowanie operatora `+` z `__add__` |
| `05_abstract_classes.py` | Klasy abstrakcyjne - ABC i `@abstractmethod` |

## Uruchomienie

```bash
python 01_polygon_inheritance.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
