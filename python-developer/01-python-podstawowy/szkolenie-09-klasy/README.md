# Szkolenie 9 - Klasy, podstawy OOP

Przykłady kodu z materiału szkoleniowego dotyczącego klas i programowania obiektowego.

## Pliki

| Plik | Opis |
|------|------|
| `01_class_basics.py` | Klasa Vehicle z konstruktorem i metodami |
| `02_encapsulation.py` | Pola prywatne, chronione, publiczne + gettery/settery |
| `03_property_decorator.py` | `@property` - pythonowy getter i setter z walidacją |
| `04_repr_vs_str.py` | `__repr__` vs `__str__` - reprezentacja obiektu |
| `05_copy_shallow_deep.py` | Kopie płytkie i głębokie obiektów |

## Uruchomienie

```bash
python 01_class_basics.py
```

Wszystkie przykłady korzystają wyłącznie z biblioteki standardowej.
