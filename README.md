# training-examples

Przykłady kodu do szkoleń [devs-mentoring.pl](https://devs-mentoring.pl).

Każdy katalog odpowiada jednemu szkoleniu i zawiera gotowy do uruchomienia kod.
Materiały teoretyczne dostępne są osobno w ramach mentoringu.

## Struktura

- `python-developer/` — przykłady ze ścieżki Python Backend Developer
  - `07-flask/` — Flask (7 szkoleń)
  - `08-fastapi/` — FastAPI (5 szkoleń)
  - ...kolejne moduły

## Jak korzystać

1. Sklonuj repozytorium: `git clone https://github.com/Devs-Mentoring-pl/training-examples.git`
2. Przejdź do katalogu interesującego Cię szkolenia
3. Przeczytaj `README.md` w katalogu, uruchom przykłady

## Wymagania

- Python 3.12+
- Zależności per szkolenie w `requirements.txt` (jeśli istnieje)
